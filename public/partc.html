Hit-and-run Indicator and Collision Severity:


This graph shows summarized accident information based on hit-and-run indicator by collision severity. It shows that when a felony is involved in the accident, the collision severity is likely to be more serious, whereas if there was no case of hit-and-run, the severity is likely to be minor, such as property damage only and complaints of pain.

Collision Type and Collision Severity:


This graph demonstrates the composition of accidents for each collision type by severity. For example, one observation is that, if the collision type is minor like a sideswipe, the collision severity is likely to be low. When a pedestrian is involved in the accidents, the collision severity is likely more serious with higher counts of fatal and severe injuries.
